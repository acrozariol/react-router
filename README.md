# React Router Navigation App

A React application built to practice client-side routing, page navigation, and dynamic routes using React Router.

## Technologies

- React
- TypeScript
- React Router
- CSS Modules
- Vite

## Features

- Multiple page routes
- Layout with shared Header and Footer
- Dynamic routes with URL parameters
- Navigation using React hooks
- Query parameters handling
- 404 Not Found page
